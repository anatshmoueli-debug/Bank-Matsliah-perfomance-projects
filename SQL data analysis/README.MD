# 📊 SQL Data Analysis | WideWorldImporters Database
---

# 📖 Project Overview

This project demonstrates advanced SQL data analysis using the **WideWorldImporters (Standard)** sample database provided by Microsoft SQL Server.

The project focuses on solving real-world business problems by writing complex SQL queries that transform raw transactional data into meaningful business insights.

The analysis covers sales performance, customer behavior, supplier performance, inventory profitability, customer retention, and business risk analysis.

The project demonstrates practical SQL skills used by Data Analysts for reporting, business intelligence, and decision support.

The main objective of this project is to apply advanced SQL techniques to solve business questions and generate analytical reports from a relational database.

# 🏢 Business Scenario

The **WideWorldImporters** database simulates the operations of a wholesale distribution company.

The database contains information about:

- Customers
- Orders
- Products
- Suppliers
- Sales
- Invoices
- Inventory
- Warehouses
- Geographic locations

Using this database, various business scenarios were analyzed to support strategic and operational decision-making.

---

# 🗄 Database

The project uses Microsoft's **WideWorldImporters (Standard)** relational database.

The analysis involves multiple related tables including:

| Table | Description |
|--------|-------------|
| Customers | Customer information |
| Orders | Customer orders |
| OrderLines | Order details |
| Invoices | Sales invoices |
| InvoiceLines | Invoice details |
| StockItems | Product information |
| Suppliers | Supplier information |
| Cities | Cities |
| Countries | Countries |
| People | Employees & Salespersons |

The project required understanding the Entity Relationship Diagram (ERD) and navigating relationships across multiple tables.
